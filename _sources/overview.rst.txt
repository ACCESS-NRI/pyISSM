Overview
========

.. warning::
   This is an experimental release. Documentation remains a work in progress. Some sections may be incomplete or under development.


pyISSM is a Python library designed to provide an interface for the `Ice-sheet and Sea level System Model (ISSM) <https://issmteam.github.io/ISSM-Documentation/>`_. It allows users to set up, run, and analyze ice sheet simulations using Python.

# pyISSM testing


This directory is home to a suite of pyISSM tests. The current test suite consists of the following key directories:

- `/assets`: Common assets used across all tests. This included data files, *.exp files and parameter files.
- `/ci_tests`: Tests automatically executing for Github CI purposes, testing core functionality of pyISSM and ISSM.
- `/general`: General tests that demonstrate pyISSM / ISSM functionality

Both `ci_tests` and `general` directories have a `dev` sub-dircetory that provides a "staging" area for in-progress tests that may not be fully-functional.
